# 开始使用 Zap Pilot

**基于市场情绪自动管理您的投资组合。** 🚀

## 🎯 Zap Pilot 适合您吗？

Zap
Pilot 专为那些希望**获得加密货币（BTC/ETH）敞口**，但希望在市场狂热期间**最小化风险**的投资者设计。

### 您应该使用 Zap Pilot，如果：

- ✅ 您相信比特币和以太坊的长期增长。
- ✅ 您在**知道何时止盈**方面遇到困难。
- ✅ 您想要一种**有纪律的、数据驱动的**投资方法。
- ✅ 您希望完全掌控自己的资金。

### 这不适合您，如果：

- ❌ 您想“梭哈”模因币（高风险）。
- ❌ 您想要保证的固定收益（尝试储蓄账户）。
- ❌ 您想手动进行日内交易。

---

## ⚡ 快速入门清单

在开始之前，请确保您拥有：

- [ ] **加密货币钱包** (MetaMask, WalletConnect 等)
- [ ] **至少 $100** 的 ETH 或稳定币 (USDC/USDT)
- [ ] **5 分钟** 完成设置

---

## 🚀 启动步骤

1.  **连接钱包**：我们支持所有主流 Web3 钱包。
2.  **存入资金**：智能合约将根据**当前市场周期**将您的资金路由到最佳配置中。
3.  **保持关注**：当市场情绪发生转变（例如，从恐惧转为中性）时，我们会通知您，提示进行再平衡。
