# 🤖 介绍

Zap Pilot 是您的自动 DeFi 投资组合管理器。

## ⚠️ 核心问题

在加密货币市场中赚钱很难。持有者往往面临：

- **没有时间**: 无法全天候监控市场。
- **情绪化决策**: 恐惧时卖出，贪婪时买入（低卖高买）。
- **高昂的 Gas 费**: 频繁的小额交易会吞噬利润。
- **复杂的钱包管理**: 在多个链上管理代币非常混乱。

## 🚀 Zap Pilot 解决方案

**“恐惧时买入。贪婪时防守。”**

Zap Pilot 使用我们的**“基于市场周期的再平衡”**（Regime-Based
Rebalancing）策略，使这一经典的投资建议自动化。

我们监控**恐惧与贪婪指数**（Fear & Greed
Index），并在市场极端情绪下为您自动调整 BTC/ETH 和稳定币的配置。

---

## 🌍 5 种市场周期（Market Regimes）

我们将市场分为 5 个独特的周期。
_注意：我们的策略是**双向**的——根据市场是升温还是降温，我们会采取不同的应对措施。_

| 周期 (Regime)                | 指标         | 策略                                                |
| :--------------------------- | :----------- | :-------------------------------------------------- |
| **极端恐惧** (Extreme Fear)  | 🔴 FGI < 25  | **最大化积累** (Maximum Accumulation)               |
| **恐惧** (Fear)              | 🟠 FGI 25-45 | **复苏** (Recovery) 或 **退出流动性池** (Unwind LP) |
| **中性** (Neutral)           | 🟡 FGI 46-54 | **假日模式 (按兵不动)** (Holiday Mode)              |
| **贪婪** (Greed)             | 🟢 FGI 55-75 | **锁定收益** (Lock Gains) 或 **持有** (Hold)        |
| **极端贪婪** (Extreme Greed) | 🟩 FGI > 75  | **最大化止盈** (Maximum Profit Taking)              |

## ✨ 为什么选择 Zap Pilot？

### 1. 它是基于情绪的

大多数机器人只关注价格。我们关注**市场心理**。

### 2. 它是非托管的

**资金永远留在您的钱包中。**
我们使用 EIP-7702 和账户抽象（AA）技术。我们提出交易，**您签署交易**。我们无法提取您的资金。

### 3. 它是跨链的

我们在以太坊、Arbitrum、Optimism、Base 和 Polygon 上为您寻找收益。

### 4. 它是“全天候”的

无论熊市还是牛市，我们的策略都能适应。我们不仅仅是在定投；我们是在**低买高卖**。
