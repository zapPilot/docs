# 🚀 Zap Pilot 如何运作

## 🧠 核心机制

Zap Pilot 是通过**市场情绪驱动的投资组合自动驾驶仪**。我们利用**恐惧与贪婪指数（Fear & Greed
Index）**，自动判断您的投资组合中应该有多少**加密货币（BTC/ETH）**与**稳定币**。

### 关键原则

- **在恐惧中买入**：当市场恐惧时，我们增加加密货币配置。
- **在贪婪时防守**：当市场狂热时，我们获利了结转入稳定币。
- **非托管**：每一次操作由您签署。我们从不持有您的资金。

---

## 1️⃣ 连接 & 存款

您连接您的钱包（MetaMask 等）并存入资金。与其选择一个随机策略，您实际上是在加入我们的**周期模型（Regime-Based
Model）**。

---

## 2️⃣ 市场周期

您的资金将根据 5 个市场周期进行分配：

| 周期 (Regime) | 指标 (Indicator) | 策略 (Strategy)               |
| :------------ | :--------------- | :---------------------------- |
| **极端恐惧**  | 🔴 FGI < 25      | **70% 加密货币** / 30% 稳定币 |
| **恐惧**      | 🟠 FGI 25-45     | **60% 加密货币** / 40% 稳定币 |
| **中性**      | 🟡 FGI 46-54     | **50% 加密货币** / 50% 稳定币 |
| **贪婪**      | 🟢 FGI 55-75     | **40% 加密货币** / 60% 稳定币 |
| **极端贪婪**  | 🟩 FGI > 75      | **30% 加密货币** / 70% 稳定币 |

**了解更多：** [详细策略指南](../strategies)

---

## 3️⃣ 📐 凯利配置 (Kelly Allocation)

为了优化我们使用的*具体*协议（例如，稳定币用 Aave 还是 Compound，LP 用 Uniswap 还是 Curve），我们使用**凯利公式（Kelly
Criterion）**。这确保我们在最大化收益的同时最小化智能合约风险。

**详细解释：** [深入了解凯利公式](../strategies/kelly-criterion)

---

## 4️⃣ ♻️ 再平衡（用户驱动）

当市场周期发生转变（例如，恐惧 → 中性）时，我们会生成一个**再平衡提案**。

1.  **通知**：您会收到警报（邮件/日历）。
2.  **执行**：您在 Zap Pilot 仪表板上点击“再平衡”。
3.  **逐步执行**：逻辑将交易分散在 5-10 天内执行，以最小化滑点和择时风险。

---

## 🔒 安全与透明

- **非托管**：您的资金始终保留在您的钱包中。
- **链上执行**：所有交易都是透明的。
- **用户控制**：您批准每一个动作。
