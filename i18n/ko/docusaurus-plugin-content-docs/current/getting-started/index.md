# Zap Pilot 시작하기

**10분 이내에 제로에서 DeFi 투자까지** 🚀

이 가이드는 지갑 설정부터 수익 성장 관찰까지 Zap Pilot에서의 첫 투자를 안내합니다.

## ⚡ 빠른 시작 체크리스트

시작하기 전에 다음을 확인하세요:

- [ ] **암호화폐 지갑** (MetaMask, WalletConnect 등)
- [ ] **일부 암호화폐 자금** (시작하려면 최소 50달러 권장)
- [ ] **5-10분** 설정 완료
- [ ] **위험 허용 범위에 대한 기본 이해**

## 🎯 1단계: 전략 선택

### 위험 평가 받기

어떤 전략이 자신에게 맞는지 확실하지 않다면, 2분 평가를 받아보세요:

**질문 1: 당신의 주요 투자 목표는 무엇인가요?**

- 🏦 고수익 예금과 유사한 안정적인 수익 창출 → **스테이블코인 볼트**
- 📈 암호화폐 시장 노출을 통해 부를 성장시키기 → **인덱스 펀드**
- ₿ 비트코인에 집중하고 향상된 수익 얻기 → **비트코인 전략**
- Ξ 이더리움 생태계 참여 → **이더리움 전략**
- ⚙️ 최대한의 제어 및 맞춤화 → **맞춤형 전략**

**질문 2: 얼마나 많은 변동성을 감당할 수 있나요?**

- 😌 최소 (안정적인 가치 선호) → **스테이블코인 볼트**
- 🙂 중간 (어느 정도의 등락은 괜찮음) → **인덱스 펀드**
- 😎 상당 (성장 잠재력을 위해 파도를 타는 것을 기꺼이 감수) → **비트코인/이더리움**
- 🤠 최대 (흥분을 환영) → **맞춤형 전략**

**질문 3: 언제 자금에 접근해야 할 수도 있나요?**

- 🏃 언제든지 예고 없이 → **스테이블코인 볼트**
- 📅 몇 달 이내 → **인덱스 펀드**
- 🗓️ 1년 이상 → **비트코인/이더리움**
- ⏰ 성과에 따라 유연하게 → **맞춤형 전략**

## 🔗 2단계: 지갑 연결

### 지원되는 지갑

모든 주요 지갑 유형을 지원합니다:

#### **브라우저 확장 프로그램**

- **MetaMask** (초보자에게 권장)
- **Coinbase Wallet**
- **Rainbow Wallet**
- **Brave Wallet**

#### **모바일 지갑**

- **WalletConnect** (200개 이상의 모바일 지갑 연결)
- **Trust Wallet**
- **Exodus**
- **Phantom** (솔라나 지원용)

#### **하드웨어 지갑**

- **Ledger** (MetaMask 또는 WalletConnect를 통해)
- **Trezor** (MetaMask를 통해)
- **콜드 스토리지** 보안을 위해 지원됩니다

### 연결 과정

1. 앱에서 **"지갑 연결" 클릭**
2. 목록에서 **지갑 유형 선택**
3. 지갑에서 **연결 승인**
4. **주소가 올바르게 표시되는지 확인**

### 첫 지갑 설정

아직 지갑이 없으신가요? 시작하는 방법은 다음과 같습니다:

#### **MetaMask 설치 (권장)**

1. **다운로드**: [metamask.io](https://metamask.io)로 이동하여 설치
2. **지갑 생성**: 설정 마법사 따르기
3. **시드 보호**: 시드 구문을 안전하게 기록
4. **네트워크 추가**: 지원되는 네트워크 추가를 도와드립니다
5. **지갑 자금 조달**: 거래소에서 암호화폐를 전송하거나 직접 구매

#### **보안 모범 사례**

- 🔐 **절대 공유하지 마세요** 시드 구문을 누구에게도
- 💾 **백업** 시드 구문을 여러 안전한 위치에
- 🔒 **강력한 비밀번호 사용** 지갑 접근을 위해
- 🚫 민감한 정보를 입력하기 전에 **URL 확인**

## 💰 3단계: 투자 자금 조달

### 최소 투자 금액

- **스테이블코인 볼트**: 최소 50달러
- **인덱스 펀드**: 최소 100달러
- **비트코인 전략**: 최소 100달러
- **이더리움 전략**: 최소 100달러
- **맞춤형 전략**: 최소 200달러

### 지원되는 자산

다음 자산 중 하나로 시작할 수 있습니다:

#### **스테이블코인**

- **USDC** (권장 - 가장 유동적)
- **USDT**
- **DAI**
- **FRAX**

#### **주요 암호화폐**

- **ETH** (이더리움)
- **WBTC** (랩핑된 비트코인)
- **BNB**
- **MATIC**

#### **지원되는 모든 체인에서**

- 선도적인 레이어 2, 이더리움, 솔라나를 지원합니다.

### 지갑에 자금 넣기

#### **중앙 집중식 거래소에서**

1. 거래소에서 **암호화폐 구매** (Coinbase, Binance 등)
2. 지갑 주소로 **인출**
3. **네트워크가 일치하는지 확인** (이더리움, Arbitrum 등)
4. **확인 대기** (일반적으로 1-5분)

#### **직접 구매** (출시 예정)

- Zap Pilot 내에서 직접 암호화폐 구매
- 신용카드 또는 은행 송금 지원
- 전략에 즉시 배포

#### **크로스체인 브릿지**

- 이미 다른 체인에 암호화폐가 있나요?
- 최적의 네트워크로 브릿지하는 것을 도와드립니다
- 인터페이스에 자동 브릿지 통합

## 🎛️ 4단계: 전략 구성

### 기본 구성

#### **투자 금액**

- 투자하려는 USD 금액 입력
- 필요한 정확한 토큰 수량 표시
- 가스 요금 추정 및 표시

#### **위험 수준** (해당 전략의 경우)

- **보수적**: 낮은 위험, 더 안정적인 할당
- **균형**: 기본 중간 위험 접근 방식
- **공격적**: 더 큰 성장 잠재력을 위한 더 높은 위험

#### **리밸런싱 빈도**

- **매주**: 최대 최적화 (비용이 약간 더 높음)
- **매월**: 균형 잡힌 접근 방식 (권장)
- **분기별**: 비용이 낮고 최적화 빈도가 낮음

### 고급 구성 (선택 사항)

#### **체인 선호도**

- **Base**: 최대 보안 및 유동성
- **Arbitrum**: 낮은 수수료, 빠른 실행
- **OP**: 가장 저렴한 수수료, 소액에 적합
- **멀티체인**: 체인 간 자동 최적화

#### **알림 선호도**

- **성과 알림**: 주요 이득/손실
- **리밸런싱 알림**: 전략 조정 시
- **시장 업데이트**: 전략에 영향을 미치는 중요한 뉴스
- **주간 보고서**: 성과 요약

## ✅ 5단계: 전략 배포

### 배포 전 검토

확인하기 전에 다음을 검토하세요:

- ✅ **전략 세부 정보** 및 예상 수익
- ✅ **투자 금액** 및 자산 할당
- ✅ **수수료 구조** 및 총 비용
- ✅ **위험 수준**이 편안함과 일치하는지
- ✅ **지갑 잔액**이 투자 + 가스에 충분한지

### 트랜잭션 승인 프로세스

2-3개의 트랜잭션을 승인해야 합니다:

#### **1. 토큰 승인** (필요한 경우)

- **목적**: Zap Pilot이 토큰을 이동할 수 있도록 허용
- **비용**: 가스 수수료 약 5-15달러
- **1회**: 토큰당, 체인당

#### **2. 전략 배포**

- **목적**: 개인화된 전략 배포
- **비용**: 가스 수수료 약 10-30달러
- **실행**: 승인 후 자동으로 발생

#### **3. 초기 할당** (필요한 경우)

- **목적**: 프로토콜 간 자금 분배
- **비용**: 가스 수수료 약 5-20달러
- **타임라인**: 일반적으로 10분 이내 완료

### 다음은 무엇인가요?

#### **즉시 (0-10분)**

- ✅ 전략 계약 배포
- ✅ 프로토콜에 자금 할당
- ✅ 실시간 추적 기능이 있는 대시보드 활성화

#### **24시간 이내**

- 📊 첫 번째 성과 데이터 사용 가능
- 🔄 필요한 경우 초기 리밸런싱
- 📧 전략 세부 정보가 포함된 환영 이메일

#### **7일 이내**

- 💰 첫 번째 수익 발생 확인 가능
- 📈 벤치마크 대비 성과 추적
- 🎯 유익한 경우 전략 최적화

## 📱 6단계: 투자 모니터링

### 대시보드 기능

#### **포트폴리오 개요**

- **현재 가치**: 실시간 USD 가치
- **총 수익**: 시작 이후의 이익/손실
- **APY 추적**: 연간 수익률
- **할당 보기**: 자금이 어디에서 작동하는지

#### **성과 분석**

- **차트**: 벤치마크와 함께 시간 경과에 따른 가치
- **지표**: 샤프 비율, 변동성, 최대 낙폭
- **귀속**: 어떤 자산/프로토콜이 수익을 주도했는지
- **예측**: 현재 금리 기반 예상 수익

#### **거래 내역**

- **모든 작업**: 입금, 인출, 리밸런싱
- **가스 비용**: 지불된 총 수수료 및 최적화
- **프로토콜 상호 작용**: 사용된 DeFi 프로토콜
- **세금 보고**: 다운로드 가능한 세금 보고서

### 모바일 앱 (출시 예정)

- 📱 **iOS 및 Android** 앱
- 🔔 중요한 이벤트에 대한 **푸시 알림**
- 📊 모바일에서 **전체 대시보드** 기능
- 🔒 추가 보호를 위한 **생체 인식 보안**

#### **"잔액 표시 안 됨"**

- **원인**: 네트워크 동기화 지연 또는 잘못된 체인
- **해결책**: 5분 기다린 후 올바른 네트워크 확인
