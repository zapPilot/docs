# 📊 策略 (Strategies)

Zap Pilot 使用單一的、有凝聚力的策略來適應市場條件。我們稱之為**“基於週期的再平衡”**（Regime-Based
Rebalancing）。

我們不要求您在“保守”或“激進”的金庫之間做出選擇，而是分析市場的心理狀態——**恐懼與貪婪指數**（Fear &
Greed Index）——並動態調整您的風險敞口。

---

## 5 種市場週期 (Market Regimes)

我們將市場分為 5 個獨特的週期。您的投資組合配置完全取決於當前市場處於哪個週期。

## 動量至關重要：雙向策略

至關重要的是，**背景很重要**。最佳舉措不僅取決於你*在哪*，還取決於你*從哪裏來*。我們稱之為**雙向邏輯**（Bi-Directional
Logic）。

### 🔴 極端恐懼 (Extreme Fear) (FGI < 25)

**“最大化積累”** (Maximum Accumulation)

- **場景**: 市場崩盤。
- **行動**: 用穩定幣積極買入 BTC/ETH。
- **配置**: **70% 加密貨幣 / 30% 穩定幣**

### 🟠 恐懼 (Fear) (FGI 25-45)

在這裏，我們的行動取決於趨勢：

- **📉 跌入恐懼 (來自中性)**:
  - **策略**: _退出流動性池轉向現貨_ (Unwind LP for Spot)。
  - **原因**: 波動性正在上升。我們消除流動性池風險（無常損失），並轉向純現貨 BTC/ETH 以捕捉反彈。
- **📈 恢復至恐懼 (來自極端恐懼)**:
  - **策略**: _監控復甦_ (Monitor Recovery)。
  - **原因**: 我們剛剛在“極端恐懼”中抄底。現在我們持有。我們還不賣出；我們讓復甦繼續。

### 🟡 中性 (Neutral) (FGI 46-54)

**“假日模式”** (Holiday Mode)

- **行動**: 什麼都不做。
- **配置**: **50% 加密貨幣 / 50% 穩定幣**
- **原因**: 市場方向不明。在這裏過度交易會吞噬費用。我們觀望並等待。

### 🟢 貪婪 (Greed) (FGI 55-75)

同樣，方向是關鍵：

- **📈 升至貪婪 (來自中性)**:
  - **策略**: _將收益鎖定到流動性池_ (Lock Gains into LP)。
  - **原因**: 牛市正在升温。我們將部分現貨比特幣輪換到**流動性池**中，在鎖定部分賬面利潤的同時，從瘋狂的交易中賺取費用。
- **📉 冷卻至貪婪 (來自極端貪婪)**:
  - **策略**: _休息一下_ (Take a Rest)。
  - **原因**: 我們已經在“極端貪婪”中賣出了頂部。隨着市場回落，我們不會立即買回。我們等待更深的回調。

### 🟩 極端貪婪 (Extreme Greed) (FGI > 75)

**“最大化防守”** (Maximum Defense)

- **場景**: 市場狂熱 / 頂部信號。
- **行動**: 大量賣出並持有穩定幣。
- **配置**: **30% 加密貨幣 / 70% 穩定幣**

## ⚙️ 再平衡如何運作

1.  **週期變更**: 當恐懼與貪婪指數從一個週期移動到另一個週期（例如，恐懼 → 中性）時，會觸發再平衡事件。
2.  **通知**: 您會收到一條通知（或日曆邀請），建議新的配置。
3.  **逐步執行**:
    - 為了防止賣得太早或買得太晚，我們不會一次性執行所有操作。
    - 我們使用**阻尼執行模型**，將交易分散在 **5-10 天**內進行。
    - 這種“平滑再平衡”確保您獲得更好的平均價格並避免由鋸齒狀波動帶來的損失。

## 🤝 託管與安全

關鍵是，**Zap Pilot 永遠不持有您的資金**。

- 所有再平衡交易均由我們提議，但**由您簽署**。
- 資金直接從您的錢包轉移到 DEX/協議並返回。
- 我們使用開源智能合約和標準協議（Uniswap, Aave 等）。
