# 開始使用 Zap Pilot

**基於市場情緒自動管理您的投資組合。** 🚀

## 🎯 Zap Pilot 適合您嗎？

Zap
Pilot 專為那些希望**獲得加密貨幣（BTC/ETH）敞口**，但希望在市場狂熱期間**最小化風險**的投資者設計。

### 您應該使用 Zap Pilot，如果：

- ✅ 您相信比特幣和以太坊的長期增長。
- ✅ 您在**知道何時止盈**方面遇到困難。
- ✅ 您想要一種**有紀律的、數據驅動的**投資方法。
- ✅ 您希望完全掌控自己的資金。

### 這不適合您，如果：

- ❌ 您想“梭哈”模因幣（高風險）。
- ❌ 您想要保證的固定收益（嘗試儲蓄賬户）。
- ❌ 您想手動進行日內交易。

---

## ⚡ 快速入門清單

在開始之前，請確保您擁有：

- [ ] **加密貨幣錢包** (MetaMask, WalletConnect 等)
- [ ] **至少 $100** 的 ETH 或穩定幣 (USDC/USDT)
- [ ] **5 分鐘** 完成設置

---

## 🚀 啓動步驟

1.  **連接錢包**：我們支持所有主流 Web3 錢包。
2.  **存入資金**：智能合約將根據**當前市場週期**將您的資金路由到最佳配置中。
3.  **保持關注**：當市場情緒發生轉變（例如，從恐懼轉為中性）時，我們會通知您，提示進行再平衡。
