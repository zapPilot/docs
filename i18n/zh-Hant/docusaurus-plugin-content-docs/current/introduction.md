# 🤖 介紹

Zap Pilot 是您的自動 DeFi 投資組合管理器。

## ⚠️ 核心問題

在加密貨幣市場中賺錢很難。持有者往往面臨：

- **沒有時間**: 無法全天候監控市場。
- **情緒化決策**: 恐懼時賣出，貪婪時買入（低賣高買）。
- **高昂的 Gas 費**: 頻繁的小額交易會吞噬利潤。
- **複雜的錢包管理**: 在多個鏈上管理代幣非常混亂。

## 🚀 Zap Pilot 解決方案

**“恐懼時買入。貪婪時防守。”**

Zap Pilot 使用我們的**“基於市場週期的再平衡”**（Regime-Based
Rebalancing）策略，使這一經典的投資建議自動化。

我們監控**恐懼與貪婪指數**（Fear & Greed
Index），並在市場極端情緒下為您自動調整 BTC/ETH 和穩定幣的配置。

---

## 🌍 5 種市場週期（Market Regimes）

我們將市場分為 5 個獨特的週期。
_注意：我們的策略是**雙向**的——根據市場是升温還是降温，我們會採取不同的應對措施。_

| 週期 (Regime)                | 指標         | 策略                                                |
| :--------------------------- | :----------- | :-------------------------------------------------- |
| **極端恐懼** (Extreme Fear)  | 🔴 FGI < 25  | **最大化積累** (Maximum Accumulation)               |
| **恐懼** (Fear)              | 🟠 FGI 25-45 | **復甦** (Recovery) 或 **退出流動性池** (Unwind LP) |
| **中性** (Neutral)           | 🟡 FGI 46-54 | **假日模式 (按兵不動)** (Holiday Mode)              |
| **貪婪** (Greed)             | 🟢 FGI 55-75 | **鎖定收益** (Lock Gains) 或 **持有** (Hold)        |
| **極端貪婪** (Extreme Greed) | 🟩 FGI > 75  | **最大化止盈** (Maximum Profit Taking)              |

## ✨ 為什麼選擇 Zap Pilot？

### 1. 它是基於情緒的

大多數機器人只關注價格。我們關注**市場心理**。

### 2. 它是非託管的

**資金永遠留在您的錢包中。**
我們使用 EIP-7702 和賬户抽象（AA）技術。我們提出交易，**您簽署交易**。我們無法提取您的資金。

### 3. 它是跨鏈的

我們在以太坊、Arbitrum、Optimism、Base 和 Polygon 上為您尋找收益。

### 4. 它是“全天候”的

無論熊市還是牛市，我們的策略都能適應。我們不僅僅是在定投；我們是在**低買高賣**。
