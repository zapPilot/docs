# 投資戦略の概要

Zap
Pilotは、さまざまなリスク選好度と投資目標に合わせて設計された、慎重に作成された投資戦略を提供します。各戦略は専門的に管理され、自動的にリバランスされ、最高のリスク調整後リターンに最適化されています。

## 📊 戦略の比較

| 戦略                                        | リスクレベル | 目標APY | 最低投資額 | 流動性   | 最適な対象             |
| ------------------------------------------- | ------------ | ------- | ---------- | -------- | ---------------------- |
| 🏦 [ステーブルコインボールト](./stablecoin) | 🟢 低        | 15-25%  | $50        | 即時     | 保守的な投資家         |
| 📈 [インデックスファンド](./index-funds)    | 🟡 中        | 15-25%  | $100       | 同日     | 分散型エクスポージャー |
| ₿ [ビットコイン戦略](./bitcoin)             | 🟠 中高      | 10-30%  | $100       | 同日     | ビットコイン信者       |
| Ξ [イーサリアム戦略](./ethereum)            | 🟠 中高      | 12-35%  | $100       | 同日     | ETHエコシステムファン  |
| ⚙️ [カスタム戦略](./custom)                 | 🔴 変動      | 変動    | $100       | 設定可能 | 上級ユーザー           |

## 🎯 戦略を選択する

### リスク許容度別

#### 🟢 **保守的（低リスク）**

**高いリターンよりも安定性を優先します**

- 主な関心事：資本保全
- 許容ボラティリティ：5％未満
- 時間軸：任意
- **推奨**：[ステーブルコインボールト](./stablecoin)

#### 🟡 **中程度（中リスク）**

**合理的なボラティリティで成長を望みます**

- リスクとリターンのバランスの取れたアプローチ
- 許容ボラティリティ：5-20％
- 時間軸：6ヶ月以上
- **推奨**：[インデックスファンド](./index-funds)

#### 🟠 **積極的（中高リスク）**

**より高いリターンのためにボラティリティを受け入れます**

- リスク管理を伴う成長重視
- 許容ボラティリティ：20-50％
- 時間軸：1年以上
- **推奨**：[ビットコイン](./bitcoin)または[イーサリアム](./ethereum)戦略

#### 🔴 **投機的（高リスク）**

**最大限のコントロールとリターンを望みます**

- 特定の目標を持つ洗練された投資家
- 許容ボラティリティ：50％以上
- 時間軸：変動
- **推奨**：[カスタム戦略](./custom)

### 投資目標別

#### 💰 **安定した収入**

暗号資産から一貫したキャッシュフローを生み出す

- **最適な戦略**：ステーブルコインボールト
- **期待利回り**：15-25％ APY
- **支払い頻度**：毎日の複利

#### 📈 **長期的な成長**

暗号市場の上昇を通じて富を築く

- **最適な戦略**：インデックスファンドまたは単一資産
- **期待リターン**：15-35％ APY
- **成長スタイル**：資本増価+利回り

#### 🛡️ **富の保全**

最小限のリスクで購買力を保護する

- **最適な戦略**：インフレヘッジ付きのステーブルコインボールト
- **期待リターン**：インフレ+ 3-8％
- **保護**：DeFi利回りによる安定した価値

#### 🔬 **実験**

新しい戦略をテストし、パフォーマンスを最適化する

- **最適な戦略**：カスタム戦略
- **期待リターン**：非常に変動性が高い
- **学習**：実践的なDeFi体験

### 時間軸別

#### ⚡ **短期（< 6ヶ月）**

柔軟性と迅速な資金アクセスが必要

- **推奨**：ステーブルコインボールト
- **流動性**：即時引き出し
- **ボラティリティ**：最小限

#### 📅 **中期（6ヶ月〜2年）**

より良いリターンのために多少のボラティリティは許容できる

- **推奨**：インデックスファンド
- **流動性**：同日引き出し
- **ボラティリティ**：中程度

#### 🏗️ **長期（2年以上）**

ボラティリティを乗り切る時間があるため、成長の可能性を最大化する

- **推奨**：ビットコイン/イーサリアムまたはカスタム
- **流動性**：同日引き出し
- **ボラティリティ**：高いが、時間がリターンを平滑化する

## 🧠 戦略の特徴

### ✅ **すべての戦略に含まれるもの**

#### **専門家による管理**

- 戦略チームによる継続的な監視
- リスク管理とポジションサイジング
- プロトコルのデューデリジェンスとセキュリティ評価
- パフォーマンスの最適化とレポート

#### **リバランス**

- 目標配分に基づく体系的なリバランス
- 取引コストを最小化するための逸脱しきい値
- 税効率の高いリバランス戦略
- 市場状況の調整

#### **リスクコントロール**

- ポジション制限と分散要件
- ストップロスとドローダウンコントロール
- プロトコルリスクの監視とアラート
- 緊急出口手順

#### **透明性**

- リアルタイムのポートフォリオ追跡
- 詳細なパフォーマンスの帰属
- 完全な取引履歴
- リスクメトリクスと分析

### 🎛️ **カスタマイズオプション**

標準的な戦略でもカスタマイズが可能です：

#### **リスク調整**

- 保守的、バランス型、または積極的なバリエーション
- カスタムのリスク制限と制約
- ボラティリティターゲティングオプション

#### **配分設定**

- チェーン設定（イーサリアム、Arbitrumなど）
- プロトコル設定（Aave、Compoundなど）
- 戦略内の資産設定

#### **リバランス設定**

- 頻度設定（毎日、毎週、毎月）
- 逸脱しきい値（3％、5％、10％）
- 市場タイミングの考慮

## 📈 パフォーマンスと分析

### 戦略メトリクス

各戦略は詳細な分析を提供します：

#### **リターン**

- 毎日、毎週、毎月、および年率換算リターン
- ベンチマーク比較
- リスク調整後パフォーマンス（シャープレシオ）

#### **リスク**

- ボラティリティと最大ドローダウン
- バリューアットリスク（VaR）計算
- 主要資産との相関

#### **コスト**

- 管理手数料と成功報酬
- 取引コストとスリッページ
- ガス最適化による節約

### 過去のパフォーマンス

バックテストおよびライブパフォーマンスを表示：

- 過去のデータに基づく**バックテスト**
- 戦略開始からの**ライブ追跡**
- さまざまな市場状況下での**シナリオ分析**
- 市場のボラティリティ中の**ストレステスト**

## 🚀 始め方

### ステップ1：戦略の選択

1. [リスク評価](../getting-started/risk-assessment)を完了する
2. 戦略の詳細とパフォーマンスを確認する
3. 配分額を選択する

### ステップ2：設定

1. 必要に応じてリスクパラメータをカスタマイズする
2. 自動入金を設定する（オプション）
3. 通知とアラートを設定する

### ステップ3：実行

1. ウォレットを接続する
2. 戦略アクセスを承認する
3. 初期投資を確認する
4. ダッシュボードでパフォーマンスを監視する
