# 📊 戦略 (Strategies)

Zap
Pilotは、市場の状況に適応する単一の一貫した戦略を使用しています。私たちはこれを**「レジーム・ベース・リバランシング（Regime-Based
Rebalancing）」**と呼んでいます。

「保守的」または「積極的」なボールトを選択するよう求める代わりに、私たちは市場の心理状態—**恐怖・強欲指数（Fear
& Greed Index）**—を分析し、動的にエクスポージャー（保有比率）を調整します。

---

## 5つの市場レジーム (5 Market Regimes)

私たちは市場を5つの異なるレジームに分類しています。あなたのポートフォリオ配分は、現在市場がどのレジームにあるかに完全に依存します。

## モメンタムが重要：双方向戦略

重要なのは、**文脈（コンテキスト）が重要だということ**です。最善の手は、あなたが*どこにいるか*だけでなく、あなたが*どこから来たか*にも依存します。私たちはこれを**双方向ロジック（Bi-Directional
Logic）**と呼んでいます。

### 🔴 極端な恐怖 (Extreme Fear) (FGI < 25)

**「積極的な蓄積」** (Maximum Accumulation)

- **シナリオ**: 市場の暴落。
- **アクション**: ステーブルコインでBTC/ETHを積極的に購入する。
- **配分**: **70% 暗号資産 / 30% ステーブルコイン**

### 🟠 恐怖 (Fear) (FGI 25-45)

ここでのアクションはトレンドに依存します：

- **📉 恐怖への下落 (中立から)**:
  - **戦略**: _スポット（現物）のためにLPを解消_ (Unwind LP for Spot)。
  - **理由**: ボラティリティ（変動率）が上昇しています。流動性プール（LP）のリスク（変動損失）を取り除き、リバウンドを捉えるために純粋な現物BTC/ETHにシフトします。
- **📈 恐怖への回復 (極端な恐怖から)**:
  - **戦略**: _回復を監視_ (Monitor Recovery)。
  - **理由**: 私たちは「極端な恐怖」で底値を買ったばかりです。今はホールドします。まだ売りません。回復の流れに乗ります。

### 🟡 中立 (Neutral) (FGI 46-54)

**「休日モード」** (Holiday Mode)

- **アクション**: 何もしない。
- **配分**: **50% 暗号資産 / 50% ステーブルコイン**
- **理由**: 市場は方向感を欠いています。ここで過剰にトレードすると手数料がかさみます。私たちは監視し、待ちます。

### 🟢 強欲 (Greed) (FGI 55-75)

再び、方向性が鍵となります：

- **📈 強欲への上昇 (中立から)**:
  - **戦略**: _利益をLPに固定_ (Lock Gains into LP)。
  - **理由**: 強気相場が加熱しています。現物ビットコインの一部を**流動性プール**にローテーションし、帳簿上の利益を確保しつつ、取引の熱狂から手数料を稼ぎます。
- **📉 強欲への冷却 (極端な強欲から)**:
  - **戦略**: _休息をとる_ (Take a Rest)。
  - **理由**: 私たちはすでに「極端な強欲」で天井を売りました。市場が下落しても、すぐには買い戻しません。より深い調整を待ちます。

### 🟩 極端な強欲 (Extreme Greed) (FGI > 75)

**「最大の防御」** (Maximum Defense)

- **シナリオ**: 市場の陶酔 / 天井のシグナル。
- **アクション**: 大量に売ってステーブルコインにする。
- **配分**: **30% 暗号資産 / 70% ステーブルコイン**

## ⚙️ リバランシングの仕組み

1.  **レジーム変更**: 恐怖・強欲指数があるレジームから別のレジームに移動すると（例：恐怖 → 中立）、リバランシングイベントが発生します。
2.  **通知**: 新しい配分を提案する通知（またはカレンダーへの招待）が届きます。
3.  **段階的な実行**:
    - 早く売りすぎたり、遅く買いすぎたりするのを防ぐため、一度にすべてを実行することはありません。
    - **減衰実行モデル（damped execution model）**を使用し、**5～10日**かけて取引を分散させます。
    - この「スムーズなリバランス」により、より良い平均価格を確保し、乱高下による損失を防ぎます。

## 🤝 カストディ（管理）と安全性

重要なのは、**Zap Pilotはあなたの資金を決して預からない**ということです。

- すべてのリバランシング取引は私たちが提案しますが、**署名するのはあなた**です。
- 資金はあなたのウォレットからDEX/プロトコルへ直接移動し、戻ってきます。
- 私たちはオープンソースのスマートコントラクトと標準的なプロトコル（Uniswap,
  Aaveなど）を使用しています。
