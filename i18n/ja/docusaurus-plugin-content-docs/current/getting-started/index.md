# Zap Pilotを始める

**市場の感情に基づいてポートフォリオを自動化する。** 🚀

## 🎯 Zap Pilotはあなたに向いていますか？

Zap
Pilotは、**暗号資産（BTC/ETH）へのエクスポージャー**を望んでいながら、市場の熱狂時には**リスクを最小限に抑えたい**投資家向けに設計されています。

### 以下の場合、Zap Pilotを使用すべきです：

- ✅ ビットコインとイーサリアムの長期的な成長を信じている。
- ✅ **いつ利益確定すべきか**で悩んでいる。
- ✅ 投資に対して**規律あるデータ主導のアプローチ**を求めている。
- ✅ 資金の完全な管理権を維持したい。

### 以下の場合、これはあなた向けではありません：

- ❌ ミームコインに「全力投資（Ape）」したい（高リスク）。
- ❌ 保証された固定利回りが欲しい（普通預金を試してください）。
- ❌ 手動でデイトレードしたい。

## 🚀 立ち上げの手順

1.  **ウォレット接続**：すべての主要なWeb3ウォレットをサポートしています。
2.  **資金の入金**：スマートコントラクトは、**現在の市場レジーム**にとって最適な配分に資金をルーティングします。
3.  **最新情報を待つ**：市場の感情が変化する（例：恐怖から中立へ）と、リバランスを促す通知が届きます。
