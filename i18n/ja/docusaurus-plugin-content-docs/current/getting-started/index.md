# Zap Pilotの始め方

**10分以内にゼロからDeFi投資へ** 🚀

このガイドでは、ウォレットの設定からリターンの成長を見守るまで、Zap
Pilotでの最初の投資を案内します。

## ⚡ クイックスタートチェックリスト

始める前に、以下のものがあることを確認してください：

- [ ] **暗号通貨ウォレット**（MetaMask、WalletConnectなど）

## 🎯 ステップ1：戦略を選択する

### リスク評価を受ける

どの戦略が自分に合っているかわからない場合は、2分間の評価を受けてください：

**質問1：主な投資目標は何ですか？**

- 🏦 高利回りの普通預金と同様の安定したリターンを得る → **ステーブルコインボールト**
- 📈 暗号市場へのエクスポージャーを通じて富を増やす → **インデックスファンド**
- ₿ ビットコインに焦点を当て、強化されたリターンを得る → **ビットコイン戦略**
- Ξ イーサリアムエコシステムに参加する → **イーサリアム戦略**
- ⚙️ 最大限のコントロールとカスタマイズ → **カスタム戦略**

**質問2：どの程度のボラティリティに対応できますか？**

- 😌 最小限（安定した価値を好む）→ **ステーブルコインボールト**
- 🙂 中程度（多少の上下は問題ない）→ **インデックスファンド**
- 😎 大幅（波に乗ることを厭わない）→ **ビットコイン/イーサリアム**
- 🤠 最大限（興奮を歓迎する）→ **カスタム戦略**

**質問3：いつ資金にアクセスする必要がありますか？**

- 🏃 いつでも予告なしに → **ステーブルコインボールト**
- 📅 数ヶ月以内に → **インデックスファンド**
- 🗓️ 1年以上先 → **ビットコイン/イーサリアム**
- ⏰ パフォーマンスに応じて柔軟に → **カスタム戦略**

### 接続プロセス

1. アプリで**「ウォレットを接続」をクリック**
2. リストから**ウォレットタイプを選択**
3. ウォレットで**接続を承認**
4. **アドレスが正しく表示されていることを確認**

### 初めてのウォレット設定

まだウォレットをお持ちでない場合は、こちらから始めてください：

#### **MetaMaskのインストール（推奨）**

1. **ダウンロード**：[metamask.io](https://metamask.io)にアクセスしてインストール
2. **ウォレットの作成**：セットアップウィザードに従います
3. **シードフレーズの保護**：シードフレーズを安全な場所に書き留めます
4. **ネットワークの追加**：サポートされているネットワークの追加をお手伝いします
5. **ウォレットへの入金**：取引所から暗号資産を転送するか、直接購入します

#### **セキュリティのベストプラクティス**

- 🔐 **シードフレーズを誰とも共有しない**
- 💾 **シードフレーズを複数の安全な場所にバックアップ**
- 🔒 **ウォレットへのアクセスには強力なパスワードを使用**
- 🚫 機密情報を入力する前に**URLを確認**

## 🎛️ ステップ4：戦略を設定する

### 基本設定

#### **投資額**

- 投資したい米ドル額を入力
- 必要な正確なトークン量が表示されます
- ガス代が見積もられ、表示されます

#### **リスクレベル**（該当する戦略の場合）

- **保守的**：リスクが低く、より安定した配分
- **バランス**：デフォルトの中程度のリスクアプローチ
- **積極的**：より高い成長の可能性を秘めたより高いリスク

#### **リバランス頻度**

- **毎週**：最大限の最適化（コストがわずかに高い）
- **毎月**：バランスの取れたアプローチ（推奨）
- **四半期ごと**：コストが低く、最適化の頻度が低い

### 詳細設定（オプション）

#### **チェーンの好み**

- **Base**：最大限のセキュリティと流動性
- **Arbitrum**：手数料が安く、実行が速い
- **OP**：手数料が最も安く、少額に適しています
- **マルチチェーン**：チェーン間で自動的に最適化

#### **通知設定**

- **パフォーマンスアラート**：大幅な利益/損失
- **リバランス通知**：戦略が調整されたとき
- **市場の更新**：戦略に影響を与える重要なニュース
- **週次レポート**：パフォーマンスの概要

## ✅ ステップ5：戦略を展開する

### 展開前レビュー

確認する前に、以下を確認してください：

- ✅ **戦略の詳細**と期待されるリターン
- ✅ **投資額**と資産配分
- ✅ **料金体系**と総費用
- ✅ **リスクレベル**があなたの快適さと一致している
- ✅ **ウォレットの残高**が投資+ガスに十分である

### 取引承認プロセス

2〜3回の取引を承認する必要があります：

#### **1. トークンの承認**（必要な場合）

- **目的**：Zap Pilotがあなたのトークンを移動できるようにする
- **費用**：ガス代で約5〜15ドル
- **1回限り**：トークンごと、チェーンごと

#### **2. 戦略の展開**

- **目的**：パーソナライズされた戦略を展開する
- **費用**：ガス代で約10〜30ドル
- **実行**：承認後、自動的に行われます

#### **3. 初期配分**（必要な場合）

- **目的**：プロトコル間で資金を分配する
- **費用**：ガス代で約5〜20ドル
- **タイムライン**：通常10分以内に完了します

### 次に起こること

#### **即時（0〜10分）**

- ✅ 戦略契約が展開されました
- ✅ 資金がプロトコルに割り当てられました
- ✅ リアルタイム追跡でダッシュボードが有効になりました

#### **24時間以内**

- 📊 最初のパフォーマンスデータが利用可能
- 🔄 必要に応じて最初の再調整
- 📧 戦略の詳細が記載されたウェルカムメール

#### **7日以内**

- 💰 最初の利回りの発生が見える
- 📈 ベンチマークに対するパフォーマンストラッキング
- 🎯 有益な場合は戦略の最適化

## 🆘 ヘルプを得る

### 初めての一般的な問題

#### **「取引に失敗しました」**

- **原因**：ガス不足またはネットワークの混雑
- **解決策**：より高いガス制限で再試行してください
- **防止策**：ガスは自動的に見積もります

#### **「ウォレットが接続されていません」**

- **原因**：ネットワークの不一致または拡張機能の問題
- **解決策**：ページを更新し、ネットワークの選択を確認してください
- **防止策**：問題が解決しない場合は、ブラウザのキャッシュをクリアしてください

#### **「残高が表示されません」**

- **原因**：ネットワークの同期遅延または間違ったチェーン
- **解決策**：5分待って、正しいネットワークを確認してください
- **防止策**：推奨ネットワークを使用してください
