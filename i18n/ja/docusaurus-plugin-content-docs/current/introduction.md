# 🤖 はじめに

Zap Pilotは、あなたのための自動DeFiポートフォリオマネージャーです。

## ⚠️ 核心的な問題

暗号資産市場でお金を稼ぐのは困難です。保有者（ホルダー）はしばしば以下の問題に直面します：

- **時間がない**: 24時間365日、市場を監視し続けることは不可能です。
- **感情的な決断**: 恐怖で売り、強欲で買う（安値で売り、高値で買う）ことになりがちです。
- **高いガス代**: 頻繁な少額取引は利益を食いつぶします。
- **複雑なウォレット管理**: 複数のチェーンでトークンを管理するのは混乱を招きます。

## 🚀 Zap Pilotのソリューション

**「恐怖で買い、強欲で守る。」**

Zap Pilotは、独自の**「レジーム・ベース・リバランシング（Regime-Based
Rebalancing）」**戦略を用いて、この古典的な投資アドバイスを自動化します。

私たちは**恐怖・強欲指数（Fear & Greed
Index）**を監視し、市場の感情が極端な状態になったときに、自動的にBTC/ETHとステーブルコインの配分を調整します。

## 🌍 5つの市場レジーム（Market Regimes）

市場を5つの異なるレジームに分類しています。
_注意：私たちの戦略は**双方向（bi-directional）**です。市場が加熱しているのか、それとも冷え込んでいるのかによって、異なる対応を取ります。_

| レジーム                       | 指標         | 戦略                                                    |
| :----------------------------- | :----------- | :------------------------------------------------------ |
| **極端な恐怖** (Extreme Fear)  | 🔴 FGI < 25  | **積極的な蓄積** (Maximum Accumulation)                 |
| **恐怖** (Fear)                | 🟠 FGI 25-45 | **回復** (Recovery) または **LPからの撤退** (Unwind LP) |
| **中立** (Neutral)             | 🟡 FGI 46-54 | **休日モード (何もしない)** (Holiday Mode)              |
| **強欲** (Greed)               | 🟢 FGI 55-75 | **利益の固定** (Lock Gains) または **ホールド** (Hold)  |
| **極端な強欲** (Extreme Greed) | 🟩 FGI > 75  | **最大の利益確定** (Maximum Profit Taking)              |

## ✨ なぜZap Pilotなのか？

### 1. 感情に基づいている

ほとんどのボットは価格だけを見ています。私たちは**市場心理**を見ています。

### 2. ノンカストディアル（非保管型）である

**資金は常にあなたのウォレットにあります。**
私たちはEIP-7702とアカウント抽象化（AA）技術を使用しています。私たちが取引を提案し、**あなたが署名します**。私たちが資金を引き出すことはできません。

### 3. クロスチェーンである

Ethereum、Arbitrum、Optimism、Base、Polygon上であなたのための利回りを探します。

### 4. 「全天候型」である

弱気相場でも強気相場でも、私たちの戦略は適応します。単なる積立投資（DCA）ではなく、**安く買い、高く売る**のです。
